# sr-app
[![Build Status](https://travis-ci.org/LarsBergqvist/sr-app.svg?branch=master)](https://travis-ci.org/LarsBergqvist/sr-app)

An Angular 8 application that uses the Open API of Swedish Public Radio

# Usage
```
cd sr-app
yarn install
yarn start
```
